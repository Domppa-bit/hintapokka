# Hintapokka 💶

Päivän kiusallisin hintapulma. Arvaa viiden kohteen hinnat — sama pulma
kaikille, joka päivä uusi. Jaa tuloksesi spoilaamatta.

**Pelaa:** hintapokka.fi (tulossa)

## Mikä tämä on
Selaimessa pyörivä päivittäispeli Wordlen hengessä, mutta sanojen sijaan
arvataan hintoja. Teema: asiat joiden hinnasta ei yleensä puhuta ääneen
(Alkon kalleimmat, hautajaiskulut, kauneuskäsittelyt, lemmikin elinkaari...).
Kaikki hinnat ovat julkista tietoa — ei nimettyjä henkilöitä, ei laittomia
tuotteita.

## Tekniikka
- Yksi `index.html` — ei palvelinta, ei tietokantaa, ei kuluja
- Päivän pulma valitaan päivämäärästä, joten kaikki saavat saman
- Kuvat `img/`-kansiossa (ks. `img/LUE-MINUT-kuvaohje.md`)

## Kehitys
Avaa `index.html` selaimessa. Siinä kaikki. Muokkaa `ITEMS`-listaa
lisätäksesi tai korjataksesi kohteita ja hintoja.

## Tehtävälista
- [ ] Tarkista kaikki hinnat oikeista lähteistä
- [ ] Lataa kuvapankkikuvat (ks. kuvaohje)
- [ ] Julkaise (Cloudflare Pages / GitHub Pages)
- [ ] Liitä domain hintapokka.fi
- [ ] Lisää kävijäseuranta (esim. Plausible)
- [ ] Mainokset / affiliate kun liikennettä on

## Lisenssi
Koodi: vapaasti muokattavissa omaan käyttöön.
Kuvat: omat tai kuvapankeista (Unsplash/Pexels/Pixabay) lisenssiehtojen mukaan.
